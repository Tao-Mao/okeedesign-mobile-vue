# Action Sheet 上拉菜单

### 安装

:::card

```javascript
import { ActionSheet } from '@okee-uikit/m-vue';

Vue.use(ActionSheet);
```

:::

### 代码演示

:::card

```topic
标准样式
```

```desc
`ActionSheet`通过`actions`数组来定义展示的选项，数组的每一项是一个对象，对象属性见文档下方表格。
```

```html
<byted-action-sheet v-model="show" :actions="actions" @select="onSelect" />
```

```javascript
export default {
  data() {
    return {
      show: false,
      actions: [{ name: '选项' }, { name: '选项' }, { name: '选项'}, { name: '选项'}],
    };
  },

  methods: {
    onSelect(item) {
      // 点击选项时默认不会关闭菜单，可以手动关闭
      this.show = false;
      this.$toast(item.name);
    },
  },
};
```

:::

:::card

```topic
展示取消按钮
```

```desc
设置`cancelText`属性后，会在底部展示取消按钮，点击后关闭当前菜单
```

```html
<byted-action-sheet
  v-model="show"
  :actions="actions"
  cancel-text="取消"
  @select="onSelect"
  @cancel="onCancel"
/>
```

:::

:::card

```topic
展示关闭按钮
```

```desc
设置`show-close`属性后，会在顶部右侧展示关闭按钮，点击后关闭当前菜单
```

```html
<byted-action-sheet
  v-model="show"
  :actions="actions"
  :show-close="true"
  @select="onSelect"
  @cancel="onCancel"
/>
```

:::

:::card

```topic
展示描述信息
```

```desc
给`actions`对应的数组里的元素添加`subname`可以添加描述信息
```

```html
<byted-action-sheet v-model="show" :actions="actions" />

export default {
  data() {
    return {
      show: false,
      actions: [
        { name: '选项' ， subname: '描述详细信息'},
        { name: '选项' ， subname: '描述详细信息'},
        { name: '选项' ， subname: '描述详细信息'},
      ],
    };
  },
};
```

### API

:::card

```topic
Types
```

```javascript
export interface ActionSheetItem {
  /**
   * 项的标题
   */
  name: string;

  /**
   * 项的副标题
   */
  subname?: string;

  /**
   * 是否在载入中状态。
   */
  loading?: boolean;

  /**
   * 禁用。
   */
  disabled?: boolean;

  /**
   *将自定义CSS类型应用于项。
   */
  className?: string;

  /**
   * 被点击时的回调，可选。
   */
  callback?: (item: ActionSheetItem) => void;

  /**
   * 自定义项的附加内容，可覆盖 `$scopedSlots.extra`。
   */
  extra?: (props: ActionSheetExtraSlotProps) => VNode;
}

export interface ActionSheetExtraSlotProps {
  item: ActionSheetItem;
  index: number;
}
```
:::

##### Props

| 参数                   | 说明                                                                                     | 类型                | 默认值  |
| ---------------------- | ---------------------------------------------------------------------------------------- | ------------------- | ------- | ---------- |
| actions                | 菜单选项                                                                                 | `ActionSheetItem[]` | -       |
| title                  | 标题                                                                                     | `string`            | -       |
| cancel-text            | 取消按钮文字                                                                             | `string`            | -       |
| overlay                | 是否显示遮罩层                                                                           | `boolean`           | `true`  |
| show-close             | 是否在标题后边展示关闭 icon                                                              | `boolean`           | `true`  |
| align                  | 项的内容的对齐方式                                                                       | `'center'`        &#124; `'left'` | `'center'` |
| divider                | 是否显示分割线                                                                           | `boolean`           | `true`  |
| close-on-click-action  | 是否在点击选项后关闭                                                                     | `boolean`           | `false` |
| close-on-click-overlay | 是否在点击遮罩层后关闭                                                                   | `boolean`           | `true`  |
| lazy-render            | 是否在显示弹层时才渲染节点                                                               | `boolean`           | `true`  |
| duration               | 动画时长，单位秒                                                                         | `number`            | `0.2`   |
| safe-area-inset-bottom | 是否开启 iPhone X 底部安全区适配，需要在 `viewport` meta 标签中设置 `viewport-fit=cover` | `boolean`           | `true`  |

##### Events

| 事件名        | 说明                                     | 回调参数                                    |
| ------------- | ---------------------------------------- | ------------------------------------------- |
| select        | 选中选项时触发，禁用或加载状态下不会触发 | item: 选项对应的对象, index: 选择对应的索引 |
| cancel        | 取消按钮点击时触发                       | -                                           |
| click-overlay | 点击遮罩层时触发                         | -                                           |
| open          | 打开菜单时触发                           | -                                           |
| opened        | 打开菜单且动画结束后触发                 | -                                           |
| close         | 关闭菜单时触发                           | -                                           |
| closed        | 关闭菜单且动画结束后触发                 | -                                           |

##### Scoped Slots

| 名称  | 说明                 | Props 类型                |
| :---- | :------------------- | :------------------------ |
| extra | 自定义项的附加内容。 | ActionSheetExtraSlotProps |
