# 表格

表格不是你的首选。

与PC端不同，移动端的展现空间极其有限。大多数场景下，表格可以被卡片等方式完全替换掉，还拥有更好的交互。
但在展现大量结构化的数据，并要对其进行分析和比较，或进行复杂操作时，它依然是最佳的展示方式。

---

### 安装
```javascript
import { Table } from '@okee-uikit/m-vue';

Vue.use(Table);
```

## 代码演示

:::card
```topic
固定高度
```
```desc
空间有限时，表格高度固定，纵向滚动查看全部数据。
```
```html
<byted-table :columns="fixedHeightColumns" :data="data" :height="350" />
```
:::

:::card
```topic
固定列
```
```desc
纵列数据较多时使用，可以固定最左列和最右列，横向滚动查看其它数据。
```
```html
<byted-table :columns="fixedColumns" :data="data" :height="350" />
```
:::

:::card
```topic
可展开
```
```desc
横行数据较多且不需要一次性完全展示全本内容时使用，默认收起。
```
```html
<byted-table
  :columns="fixedHeightColumns"
  :data="data"
  :height="350"
  :expanded="expanded"
  :expandOptions="expandingOptions"
  @expand-change="onExpandChange"
/>
```
:::


:::card
```topic
筛选和排序
```
```desc
对某一列数据进行筛选或排序，可快速查找数据。
```
```html
<byted-table
  :columns="columns"
  :data="data"
  :height="350"
  :sortDataProp="sortDataProp"
  :sortType="sortType"
  @sort-change="handleSortChange"
  @filter-change="handleFilterChange"
/>
```
:::


:::card
```topic
表格内容为空
```
```desc
当表格内容为空时显示缺省样式。
```
```html
<byted-table :columns="fixedHeightColumns" :data="[]" :height="350" />
```
:::

:::card
```topic
吸顶表格
```
```desc
表格在随页面滚动时，表头可以固定在页面顶部不消失，在表格内纵向滚动查看其它数据。
```

:::card
```topic
TS类型
```
```javascript
export interface TableProps extends React.HTMLAttributes<HTMLDivElement>, EmptyContentProps {
  columns: TableColumnProps[];
  data: TableDataProps[];

  dataPropsMapping?: TableDataPropType;

  height?: string | number;
  
  sortDataProp?: string;
  sortType?: TableSortType;
  
  expanded?: string[];
  expandOptions?: TableExpandOptionsType;

  ceilingOptions?: TableCeilingOptionsType;
  
  loading?: boolean;

  swipeable?: boolean;
  showIndicator?: boolean;
  

  onSortChange?: (column: TableColumnProps, sortDataProp: string, sortType: TableSortType) => void;
  onFilterChange?: (column: TableColumnProps, filterDataProp: string) => void;
  onExpandChange?: (expandRowData: TableDataProps, newExpanded: string[]) => void;
}

```

## API

##### Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|------|
| columns | 表格列的配置描述 | `TableColumnProps[]` | `[]` |
| data | 显示的数据 | `TableDataProps[]` | `[]` |
| height | 表格高度, 设置后,如果表格内容大于此值，会固定表头 | `string` | `null` |
| sortDataProp | 排序生效的列属性 | `string` | `null` |
| sortType | 排序生效的顺序 | `TableSortType` | `false` |
| expanded | 展开的行的id | `string[]` | `[]` |
| expandOptions | 展开的配置描述 | `TableExpandOptionsType` | `null` |
| ceilingOptions | 吸顶的配置描述，吸顶原理是使用sticky定位，将需要吸附的元素设置为 overflow 不为 visible。层级中间不能有其他设置了 overflow 的元素 | `TableCeilingOptionsType` | `null` |
| swipeable | 是否通过滑动来控制列移动 | `boolean` | `false` |

## Events

| Event | Description | Arguments |
|------|------|------|
| onSortChange | 触发排序改变 | (column: TableColumnProps, sortDataProp: string, sortType: TableSortType) => void; | event: Event |
| onFilterChange | 触发过滤改变 | (column: TableColumnProps, filterDataProp: string) => void; | event: Event |
| onExpandChange | 触发展开改变 | (expandRowData: TableDataProps, newExpanded: string[]) => void; | event: Event |
