# Steps

### Install

:::card

```javascript
import { Steps } from '@okee-uikit/m-vue';

Vue.use(Steps);
Vue.use(Step);
```

:::

### Usage

:::card

```topic
Basic Usage
```

```html
<byted-steps :current="current" @change="handleClick">
  <byted-step v-for="item in steps" :key="item.title" v-bind="item"></byted-step>
</byted-steps>
```

```javascript
export default {
  data() {
    return {
      steps: [
        {
          title: 'Completed',
        },
        {
          title: 'In Progress',
          inProgress: true,
        },
        {
          title: 'Waiting',
        },
        {
          title: 'Incompleted',
          status: 'error',
        },
      ],
    };
  },
};
```

### API

##### Props

| Attribute | Description                                                 | Default      |
| --------- | :---------------------------------------------------------- | :----------- |
| current   | Number                                                      | 1            |
| status    | String Enumerate: 'error', 'finished', 'waiting', 'inprogress' | 'inprogress' |
| direction | String Enumerate: 'horizontal', 'vertical'                     | 'horizontal' |
| onChange  | (index) => void, the step index of the item being clicked       | undefinded   |
