# Steps 步骤条

### 安装

:::card

```javascript
import { Steps } from '@okee-uikit/m-vue';

Vue.use(Steps);
Vue.use(Step);
```

:::

### 代码演示

:::card

```topic
基础用法
```

```html
<byted-steps  :current="current" @change="handleClick">
  <byted-step v-for="item in steps" :key="item.title" v-bind="item"></byted-step>
</byted-steps>
```

```javascript
export default {
  data() {
    return {
      steps: [
        {
          title: '已完成',
        },
        {
          title: '进行中',
          inProgress: true,
        },
        {
          title: '待完成',
        },
        {
          title: '未完成',
          status: 'error',
        },
      ],
    };
  },
};
```

### API

##### Props

| 参数      | 说明                                                        | 默认值       |
| --------- | :---------------------------------------------------------- | :----------- |
| current   | Number                                                      | 1            |
| status    | String 枚举值：'error', 'finished', 'waiting', 'inprogress' | 'inprogress' |
| direction | String 枚举值：'horizontal', 'vertical'                     | 'horizontal' |
| onChange  | (index) => void, index 为被点击的 step index                | undefinded   |
