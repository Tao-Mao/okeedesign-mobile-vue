# Collapse 折叠面板

### 安装

:::card

```javascript
import { Collapse, CollapseItem } from '@okee-uikit/m-vue';

Vue.use(Collapse).use(CollapseItem);
```

:::

### 代码演示

:::card

```topic
基础用法
```

```desc
通过`v-model`控制展开的面板列表，`active`为数组格式
```

```html
<byted-collapse v-model="active">
  <byted-collapse-item :title="标题" :message="内容" />
  <byted-collapse-item :title="标题" :message="内容" />
  <byted-collapse-item :title="标题" :message="内容" />
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      active: [0],
    };
  },
};
```

:::

:::card

```topic
手风琴
```

```desc
通过`accordion`可以设置为手风琴模式，最多展开一个面板，此时`active`为字符串格式
```

```html
<byted-collapse v-model="active" accordion>
  <byted-collapse-item :title="标题" :message="内容" name="1" />
  <byted-collapse-item :title="标题" :message="内容" name="2" />
  <byted-collapse-item :title="标题" :message="内容" name="3" />
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      active: '1',
    };
  },
};
```

:::

:::card

```topic
禁用状态
```

```html
<byted-collapse v-model="active">
  <byted-collapse-item :title="标题" :message="内容" />
  <byted-collapse-item disabled :title="标题" />
  <byted-collapse-item disabled :title="标题" />
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      active: [0],
    };
  },
};
```

:::

:::card

```topic
自定义标题内容
```

```html
<byted-collapse v-model="active">
  <byted-collapse-item :title="标题">
    <template #title>
      <img src="cell.svg" alt="cell" width="17px" height="17px">
      <span>标题</span>
    </template>
  </byted-collapse-item> 
  <byted-collapse-item :title="标题">
    <template #value>
      <span>内容</span>
    </template>
  </byted-collapse-item>
</byted-collapse>
```

```javascript
export default {
  data() {
    return {
      active: [],
    };
  },
};
```

### API

##### Collapse Props

| 参数      | 说明                | 类型      | 默认值   |
| --------- | ------------------- | --------- | -------- | ------ | ------- | --- |
| v-model   | 当前展开面板的 name | `string[]` &#124; `number[]` &#124; `string` &#124; `number` | -   |
| accordion | 是否开启手风琴模式  | `boolean` | `false`  |
| border    | 是否显示外边框      | `boolean` | `true`   |

##### Collapse Events

| 事件名 | 说明           | 回调参数             |
| ------ | -------------- | -------------------- | ------ |
| change | 切换面板时触发 | activeNames: `string` `array` |

##### CollapseItem Props

| 参数      | 说明                                 | 类型      | 默认值  |
| --------- | ------------------------------------ | --------- | ------- | ------- |
| name      | 唯一标识符，默认为索引值             | `string`   &#124; `number` | `index` |
| size      | 标题栏大小，可选值为 `large`         | `string`  | -       |
| title     | 标题栏左侧内容                       | `string` &#124; `number` | -       |
| value     | 标题栏右侧内容                       | `string` &#124; `number` | -       |
| label     | 标题栏描述信息                       | `string` &#124; `number` | -       |
| border    | 是否显示内边框                       | `boolean` | `true`  |
| clickable | 是否开启点击反馈                     | `boolean` | `false` |
| is-link   | 是否展示标题栏右侧箭头并开启点击反馈 | `boolean` | `true`  |
| disabled  | 是否禁用面板                         | `boolean` | `false` |

##### CollapseItem Slots

| 名称       | 说明                          |
| ---------- | ----------------------------- |
| default    | 面板内容                      |
| value      | 自定义显示内容                |
| title      | 自定义`title`                 |
| right-icon | 自定义右侧按钮，默认是`arrow` |
