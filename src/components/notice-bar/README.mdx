# NoticeBar 通知栏

### 安装

:::card

```javascript
import { NoticeBar } from '@okee-uikit/m-vue';

Vue.use(NoticeBar);
```

:::

### 代码演示

:::card

```topic
基础用法
```

```html
<byted-notice-bar type="primary" :text="reminder" />
<byted-notice-bar type="success" :text="success" />
<byted-notice-bar type="warning" :text="warning" />
<byted-notice-bar type="danger" :text="danger" />
```

:::

:::card

```topic
通知栏模式
```

```desc
运用插槽自定义其他模式
```

```html
<byted-notice-bar show-close type="warning" :text="warning" >
  <img src="horn.svg" slot="left" width="16px" height="16px" alt="horn">
</byted-notice-bar>
<byted-notice-bar  type="warning" :text="warning">        
  <byted-button slot="right" type="text">
    button
  </byted-button>
  <img src="horn.svg" slot="left" width="16px" height="16px" alt="horn">
</byted-notice-bar>
```

:::

### API

##### Props

| 参数       | 说明                                                     | 类型      | 默认值    |
| ---------- | -------------------------------------------------------- | --------- | --------- |
| text       | 通知文本内容                                             | `string`  | `''`      |
| type       | 样式类型,可设置为 `primary` `success` `warning` `danger` | `string`  | `warning` |
| show-close | 展现关闭 icon                                            | `boolean` | `false`   |
| ellipsis   | 开启文本几行打点                                         | `number`  | `0`       |

##### Events

| 事件名 | 说明             | 回调参数     |
| ------ | ---------------- | ------------ |
| click  | 点击通知栏时触发 | event: Event |
| close  | 关闭通知栏时触发 | event: Event |

##### Slots

| 名称    | 内容             |
| ------- | ---------------- |
| default | 通知文本内容     |
| left    | 文本区域左边内容 |
| right   | 文本区域右边内容 |
