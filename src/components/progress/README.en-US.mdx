# Progress

### Install

:::card

```javascript
import { Progress } from '@okee-uikit/m-vue';

Vue.use(Progress);
```

:::

### Usage

:::card

```topic
Basic Usage
```

```html
<byted-progress :percentage="60" />
<byted-progress :percentage="100" color="#6ABF40" />
```

:::

:::card

```topic
Disabled
```

```html
<byted-progress :percentage="60" :inactive="false" />
```

:::

:::card

```topic
Style
```

```html
<byted-progress :percentage="60" :text-in-middle="true" />
<byted-progress :percentage="60" color="#6ABF40" :text-in-middle="true" />
<byted-progress :percentage="100" :text-in-middle="true" />
```

:::

### API

##### Props

| Attribute    | Description                                    | Type      | Default   |
| ------------ | ---------------------------------------------- | --------- | --------- |
| percentage   | progress show percentage                       | `number`  | `50`      |
| trackColor   | progress background color                      | `string`  | `#EBEBEB` |
| color        | progress percentage show part background color | `string`  | `#338AFF` |
| inactive     | is in active                                   | `boolean` | `true`    |
| textInMiddle | is show percentage text in progress middle     | `boolean` | `false`   |
